COMMANDS:
─────────

Standard build (no protection):
  npm run build

Protected build:
  npm run build:protect

Standard deploy:
  npm run deploy

Protected deploy (RECOMMENDED):
  npm run deploy:protect

Force deploy (rare):
  npm run deploy:force

Local testing:
  npm run build:protect
  cd dist
  python -m http.server 8000
  # Visit http://localhost:8000


PROTECTION LAYERS:
──────────────────

1. Canvas Rendering     - Images rendered via canvas, not direct files
2. Invisible Overlay    - Transparent overlay blocks interactions
3. Signed URLs          - HMAC-SHA256 signatures with 1-hour TTL
4. Right-Click Block    - Global context menu disabled
5. Image Tiling         - High-value images split into 256×256 tiles


PERFORMANCE:
─────────────

Build time:     2s → 3s (+50%)
File size:      58KB → 66KB (+14%)
Gzipped:        Minimal increase (~2%)
Runtime impact: ~100ms canvas render, no noticeable lag
